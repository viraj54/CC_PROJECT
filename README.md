# Prostate Cancer Detection

This project uses a simple decision tree model to predict whether a histopathology image indicates **Cancer** or **Non-Cancer**. The model is implemented in **Flask** and predicts based on random features (for testing purposes).

## Setup

1. Clone the repository.
2. Install the dependencies using `pip install -r requirements.txt`.
3. Run the Flask application with `python app.py`.

## File Structure

- `app.py`: Flask application and decision tree logic.
- `templates/index.html`: The user interface for uploading images.
- `static/`: Folder for uploaded images.
- `requirements.txt`: Project dependencies.
